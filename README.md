# Proyectos educativos sobre Telecomunicación

En este repositorio se compilan tutoriales y proyectos que permiten familiarizarse con conceptos de Ingeniería de Telecomunicación.
El listado se va actualizando con nuevos ejemplos.

Los materiales utilizados se basan en EPS32 SteamMaker, tarjetas Micro:bit, y componentes que pueden encontrarse fácilmente.
